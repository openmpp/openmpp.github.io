OpenM++: open source microsimulation platform

OpenM++ is an open source microsimulation platform inspired by and compatible with Modgen. OpenM++, compare to it close source predecessor, has many distinct features like portability, scalability and open source. It is not a copy of the Modgen, but a new, functionally equal implementation of publicly available specifications. OpenM++ does not depend on Modgen, does not use Modgen and does not share any source code with Modgen. On the other hands, you can be compile and run Modgen models by using OpenM++.
