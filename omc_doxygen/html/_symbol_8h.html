◆ symbol_map_type

typedef unordered_map<string, Symbol *> symbol_map_type

Defines an alias representing the type of the symbol table.

The symbol table maps strings to pointers to symbols. The use of pointers to Symbols allows polymorphic symbols in the symbol table. It also allows a given symbol to be 'morphed' to a different kind of symbol during parsing by replacing the Symbol pointer by a pointer to a new instance of a derived class of Symbol. Morphing requires that any persistent use of a Symbol in another object must be through an additional level of indirection, i.e. Symbol **.
