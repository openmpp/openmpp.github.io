bool omc::skipPathModule ( const string &  path)

true if the module given by path is skipped by omc.

As a special case, module stems starting with "modgen_" are skipped by omc. This mechanism allows a single model code base to support compilation by either the Modgen or OpenM++ compilers.

Parameters
pathFull pathname of the file.
Returns
True if path is a module skipped by omc.
