Detailed Description

Symbol for the model version information.

A single instance of VersionSymbol exists in the symbol table, with name 'version'. A default VersionSymbol with value 1,0,0,0 is created at initialization. This default VersionSymbol will be morphed if a 'version' statement is encountered in the model source code.
