Detailed Description

Literal encapsulates a C++ literal or an openM++ literal.

When a C++ literal is identified by the scanner it creates an instance of a sub-class of Literal which encapsulates the C++ representation of the literal as a string, and passes it to the parser. For example, a FloatingPointLiteral might encapsulate a string consisting of the 7 characters in 2.71828.

The sub-class TimeLiteral is the only literal which is not a C++ literal.
